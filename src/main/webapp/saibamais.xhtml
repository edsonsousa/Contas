Saiba mais

O Zig é possibilita a transferência de valores entre os seus usuários de forma segura e prática.

Como Funciona

Ao se cadastrar no sistema, o usuário cria uma chave exclusiva que o identifica. Com esta Chave os usuários podem transferir valores entre si a partir de Contas Bancárias convencionais e Cartões de crédito.

Comerciantes

Para você que recebe vários pagamentos, o sistema oferece suporte para acompanhar e controlar os recebimentos.
